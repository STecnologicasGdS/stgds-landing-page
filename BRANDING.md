# Sistema de branding web

Este paquete incluye una versión optimizada del branding para uso web.

## Archivos principales

- `assets/brand-mark.png` / `brand-mark.webp`  
  Isotipo optimizado para navbar, botones, íconos y usos compactos.

- `assets/logo-header.png` / `logo-header.webp`  
  Logo horizontal transparente para encabezados o usos secundarios.

- `assets/logo-hero.png` / `logo-hero.webp`  
  Logo principal para secciones grandes, presentaciones o hero cards.

- `assets/logo-footer.png` / `logo-footer.webp`  
  Logo horizontal claro para footer.

- `assets/favicon.ico`, `favicon-32.png`, `favicon-192.png`, `favicon-512.png`  
  Favicons para navegador, móvil y accesos directos.

- `assets/apple-touch-icon.png`  
  Ícono para dispositivos Apple.

- `assets/site.webmanifest`  
  Manifest básico para identidad del sitio.

## Reglas de uso

- Navbar: usar `brand-mark.webp`
- Hero o presentación visual: usar `logo-hero.webp`
- Footer: usar `logo-footer.webp`
- No usar logos con fondo sólido dentro del navbar.
- Preferir `.webp` en la web y `.png` cuando se necesite compatibilidad o edición.
